# AI TERMINAL ASSISTANT

A command-line AI chatbot built with python using the Groq API

## Features

- Conversational Memory
- Streaming AI responses
- Persistent chat history using json
- Modular Architecture 
- Built in commands 

## Commands 

- /help
- /save 
- /clear
- /exit

## Project structure

    |--main.py    
    |--llm.py    
    |--memory.py    
    |--config.py    
    |--chat_history.json
    |_.gitignore
